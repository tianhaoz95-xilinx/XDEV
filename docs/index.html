Quick Start

XDEV is simply a CMake project consist of a lot small utilities

1. Clone the repository
2. Simply do a CMake
3. For testers: ctest
4. For developers: ./bin/awesome-tool

Why XDEV?

Developing for specialized hardware deserves a specialized deveoping framework. Fair enough.

Consider XDEV as a big party for small utilities for developing both software code and hardware code, and scripts to make life easier.
