Detailed Description

struct ert_start_kernel_cmd: ERT start kernel command format

: [3-0] current state of a command : [11-10] extra CU masks in addition to mandatory mask : [22-12] number of words in payload (data) : [27-23] 0, opcode for start_kernel : [31-27] 0, type of start_kernel

: first mandatory CU mask : count number of words representing command payload

The packet payload is comprised of 1 mandatory CU mask plus extra_cu_masks per header field, followed a CU register map of size (count - (1 + extra_cu_masks)) uint32_t words.
