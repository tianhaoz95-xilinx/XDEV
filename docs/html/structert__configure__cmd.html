Detailed Description

struct ert_configure_cmd: ERT configure command format

: [3-0] current state of a command : [22-12] number of words in payload (5 + num_cus) : [27-23] 1, opcode for configure : [31-27] 0, type of configure

: command queue slot size : number of compute units in program : shift value to convert CU idx to CU addr : base address to add to CU addr for actual physical address

:1 enable embedded HW scheduler :1 poll for command completion :1 enable CUDMA custom module for HW scheduler :1 enable CUISR custom module for HW scheduler :1 enable interrupt from host to HW scheduler :1 enable CDMA kernel :25 :1 reserved for internal use

: addresses of CUs
